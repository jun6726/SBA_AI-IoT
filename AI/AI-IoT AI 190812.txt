머신러닝 개론
 인공지능 vs 머신러닝 vs 딥러닝
 머신러닝 - 기계를 학습시키는
 딥러닝 - 머신러닝의 일종, 다층 인공신경망 구조로 빅 데이터 학습
 	ex) A와 B를 주고 +라는 연산자와 A+B를 주는게 기존 컴퓨팅이라고한다면
	     A와 B같은 입력 값과 결과 값을가지는 여러 데이터 셋을 제공하여 연산자를 추측하도록 함
 지도 학습, 감독학습 (Supervised Learning) - 문제와 정답을 제공, 가장 많이 사용하는 방법, 예측/추정/분류에 사용
 비지도 학습(Unsupervised Learning) - 문제만 제공, 평소랑 다른값만 확인하면 될 때, 정답이 필요없음, 의료분야에서 비정상세포 확인할때
				정상 데이터 값만 학습하다가 특이점이 있는 데이터를 입력 받으면 Anomaly Detection 
강화 학습 (Reinforcement Learning) - 정답이 아닌 보상 제공, 인과 관계가 중요, 게임/로봇, 

머신러닝에서는 input 데이터에 대해서 output에 영향을 끼치는 지 확인하고 필터링해야함. (데이터를 이해하고 분석전문가가 한다)
딥러닝에서하는 Feature Extraction은 기계가 내부적으로 진행 (많은 데이터와 고성능 HW가 요구됌)
